Parágrafos e Quebras de Linhas

 Você pode escrever um parágrafo de qualquer jeito. Basta colocar tudo no meio do par de tags <p>.

 Se precisar quebrar o texto em algum lugar
como esse, você pode usar a tag <br>.

HTML Entities

 Vamos adicionar alguns símbolos especiais:
 = Espaço em branco;
® = Marca Registrada;
© = Copyright;
™ = Trade Mark;
€ = Euro;
£ = Libra (pound);
¥ = Yen;
¢ = Cent;
∅ = Vazio;
∑ = Soma;
Δ = Delta;
← = Seta Esquerda;
↑ = Seta Acima;
→ = Seta Direita;
↓ = Seta Baixo.

Emojis

 Vamos adicionar alguns emojis:
😉 = Cara Piscando;
😜 = Piscando o Rosto com a Língua
🫣 = Rosto com Olho Espreitando
🤓 = Cara de Nerd
😵 = Rosto com Olhos Cruzados
